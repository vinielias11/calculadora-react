<h1 align="center">Calculadora React</h1>

![Screenshot_1](https://user-images.githubusercontent.com/83616856/154286074-2f20baef-d7a6-403a-97e6-b99a87f47c27.png)

Pequeno projeto para treinar React e CSS-grid. Consiste em um app básico de calculadora com quatro componentes.

O objetivo é permitir ao usuário fazer operações simples de adição, subtração, multiplicação e divisão. Também conta com um botão para inverter o sinal do número em tela.

<h3>Tecnologias utilizadas</h3>
<li>ReactJS</li>
<li>CSS</li>
<li>react-textfit</li> <br/>
Feita com base nesse artigo: https://www.sitepoint.com/react-tutorial-build-calculator-app/
